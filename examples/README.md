# PHP SDK for XMLY Services

## Features

- [x] 账号 API
  - [x] 获取访问令牌
  - [x] 刷新访问令牌
  - [x] 回收访问令牌
  - [x] 回收刷新访问令牌
  - [x] 查询访问令牌
- [x] 点播 API
